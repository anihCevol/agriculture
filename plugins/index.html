# 常用库

  • momentjs (opens new window)JavaScript 日期处理类库
  • elementUI (opens new window)Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库
  • Element Plus (opens new window)Element Plus，一套为开发者、设计师和产品经理准备的基于 Vue 3.0 的桌面端组件库
  • VuePress (opens new window) 本网站就是基于它实现的，简单方便的静态网站生成器
  • lodash (opens new window),中文网 (opens new window) 前端工具函数集合
  • screenfull.js (opens new window) 浏览器全屏插件 解决了不少兼容性问题
  • primjs (opens new window) 让页面支持代码高亮
  • animate.css (opens new window) 最有名的动画效果库
  • direction-reveal (opens new window) 根据鼠标进入位置，展现从不同方向 展现 hover 效果
  • Vue.Draggable (opens new window) Draggable 拖拽组件 基于 Sortable.js 的 vue 版本
  • axios (opens new window) 目前最常见的请求库
  • luch-request (opens new window) ，是一个基于Promise 开发的uni-app跨平台、项目级别的请求库
  • eruda (opens new window) 移动端调试工具,开发移动端页面，特别是调试开发线上环境
  • echarts (opens new window) 一个基于 JavaScript 的开源可视化图表库
  • vue-cropper (opens new window) 一个优雅的图片裁剪插件

TIP
